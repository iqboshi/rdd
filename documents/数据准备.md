# 数据准备阶段说明

更新时间：2026-03-06

## 1. 当前数据目录约定

- 原始图像：`data/raw/original_images/`
- 预处理后图像（用于后续训练输入）：`data/processed/preprocessed_images/`
- 训练阶段使用图像：`data/processed/images/`
- ISAT 标注：`data/processed/labels/`（`.json`）

说明：图像与标注按同名匹配，仅后缀不同。

## 2. 图像预处理（SAM 友好）

脚本：`scripts/preprocess.py`

目标：
- 减弱阴影/强光照差异
- 抑制高亮低饱和反光区域
- 保边去纹理，减少高频噪声

主要流程（按顺序）：
1. Resize（可选）
2. CLAHE（LAB 的 L 通道）
3. 双边滤波（保边平滑）
4. 高光抑制（HSV 中高 V、低 S 区域软压亮）
5. Unsharp（可选，默认关闭）

当前默认参数（已调强一档）：
- `clahe_clip=2.8`
- `clahe_grid=6`
- `bilateral_d=9`
- `bilateral_sc=80`
- `bilateral_ss=80`
- `hl_v_thresh=210`
- `hl_s_thresh=55`
- `hl_strength=0.5`

示例运行：

```bash
python scripts/preprocess.py
```

如需保存前后对比图：

```bash
python scripts/preprocess.py --save_compare
```

## 4. 关键逻辑定义

### 4.1 实例定义（ISAT）

- 使用 `group` 作为实例 ID 的逻辑单元（`group = instance`）。
- 同一 `group` 下多个分片 polygon 会先做并集，再作为一个实例。
- 若对象缺失 `group`，回退为“该 object 单独一个实例”。

### 4.2 语义图（semantic_map）

- 单通道整型图（保存为 `.npy`）。
- `0` 为背景。
- 前景类别 ID 由 `class_id_mapping.txt` 维护（从 1 开始）。

### 4.3 实例图（instance_map）

- 单通道整型图（保存为 `.npy`）。
- `0` 为背景。
- 每个实例一个唯一正整数 ID。
- 如存在重叠，按 `layer` 排序后写入，后写入实例覆盖前者。

### 4.4 中心图（center_map）

- 每个实例先计算像素质心：
  - `cy = mean(y)`
  - `cx = mean(x)`
- 在 `(cx, cy)` 位置绘制高斯峰（`sigma` 由 `--center_sigma` 控制，默认 4.0）。

### 4.5 偏移图（offset_map）

- 形状为 `H x W x 2`，`float32`。
- 对实例内部像素 `(y, x)`：
  - `offset[..., 0] = cy - y`（dy）
  - `offset[..., 1] = cx - x`（dx）
- 背景偏移为 0。

### 4.6 密度图（density_map）

- 对每个实例，面积记为 `A_i`。
- 实例内部每个像素赋值 `1 / A_i`，背景为 0。
- 全图积分（像素和）应接近实例数。
- 脚本会打印一致性检查：
  - `instances`
  - `density_sum`
  - `consistent`

## 5. 当前阶段产物总结

数据准备阶段已打通以下链路：
1. 原始图像预处理（增强可分割性）。
2. ISAT 标注解析并按 `group` 合并实例。
3. 同步生成多任务监督目标：
   - `semantic / instance / center / offset / density`
4. 生成可视化与一致性检查日志，用于快速质检。
